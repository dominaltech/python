# pyBuddy - Python IDE with AI Assistant

Built by **Team Dominal** (Yahya Mundewadi & Amir Rangrez)
- Website: https://www.dominal.in
- AI Platform: https://www.ai.dominal.in

## Features
- Run Python code in browser using Pyodide
- AI-powered coding assistant using Groq API
- Real-time error detection and suggestions
- Code explanations and optimizations
- Mobile-friendly responsive design

## Setup
1. Clone this repository
2. Get Groq API key from https://console.groq.com
3. Deploy to Netlify
4. Add GROQ_API_KEY environment variable in Netlify

## Technologies
- Pyodide (Python in browser)
- Groq API (Llama 3.3 70B model)
- Netlify Functions (serverless backend)
- Vanilla JavaScript & HTML5
